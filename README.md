# Coffee-Shop-Sales

## Project Overview

This project analyzes transactional data from a coffee shop to uncover insights into sales performance, customer purchasing behavior, and product trends.
An interactive dashboard was built in Microsoft Excel to monitor key business KPIs such as revenue, footfall, peak hours, and category contribution.

The goal is to help business stakeholders quickly identify high-performing products, busy time periods, and store-level performance.

## Business Objectives

- Identify peak ordering hours during the day.
- Determine top-selling products based on revenue.
- Compare sales and footfall across store locations.
- Analyze weekday ordering patterns.
- Understand category and size contribution to total sales.

## Tools & Skills Used

- MS Excel
- Pivot Tables & Pivot Charts
- KPI Metrics
- Data Cleaning, Transformation and aggregation
- Dashboard Design
- Business Insight Generation

## Dashboard Features

- Month-wise filter
- Day-wise filter

The analysis reveals strong morning sales concentration, high contribution from coffee products, and noticeable variation across store locations and weekdays. The dashboard provides actionable insights for improving operations, inventory planning, and marketing strategies.
